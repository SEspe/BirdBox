# BirdBox UI language cross-reference (FSD v2.72, i18n Phase 1).
#
# HOW TO EDIT
#  - TAB-separated. Column 0 = the ENGLISH master string, exactly as rendered
#    in the UI (real characters, not HTML entities: — not &mdash;). It is the
#    lookup key; if it does not match the on-screen text letter-for-letter,
#    the string silently stays English (safe fallback, nothing breaks).
#  - Leading emoji on buttons are NOT part of the key ("💾 Save Settings" is
#    keyed as "Save Settings" — the translator strips the first token when the
#    full text has no entry).
#  - Lines starting with # are comments. Blank lines ignored.
#  - After editing: rebuild + flash (the file is embedded in the firmware).
#
# ADDING A LANGUAGE
#  1. Add its code to the #cols line below (e.g. "#cols<TAB>no<TAB>de").
#  2. Add a translation column to every row (missing cells fall back to EN).
#  3. In web_server.c: add the language to the Settings dropdown and to the
#     LANGC array (index = the settings lang enum value).
#

@sinfo:sens:alt	0–100. Senk den hvis løv eller skygger utløser hendelser; øk den hvis små fugler ikke fanges opp.

@sinfo:civl:alt	250–10000 ms i steg på 250 ms.
